# 🚀 Automação de Carga de Dados de Estoque (ETL)

Este projeto automatiza o processo de extração, transformação e carga (ETL) de dados operacionais, simulando um cenário de gestão de estoque.

## 🛠️ Tecnologias Utilizadas
* **Python**: Linguagem principal para automação.
* **Pandas**: Manipulação e tratamento de dados.
* **SQLAlchemy**: Integração e carga no banco de dados.
* **MySQL**: Armazenamento estruturado dos dados.

## 📋 Funcionalidades
* **Extração**: Leitura de arquivos CSV de estoque.
* **Transformação**: Padronização de strings para garantir integridade.
* **Carga**: Inserção automática no banco de dados via Python, eliminando processos manuais.
